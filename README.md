# AI Interactive Learning Web

An interactive web-based educational platform developed to improve users' understanding of Artificial Intelligence through multimedia learning materials, presentations, and quizzes.

## Overview

This project was developed as an educational media platform that introduces fundamental Artificial Intelligence concepts in an engaging and interactive way. Users can learn through presentation slides, audio narration, videos, and quizzes to evaluate their understanding.

## Features

- Interactive learning materials about Artificial Intelligence
- Multimedia content (images, audio, and video)
- Slide-based educational presentations
- Quiz and evaluation module
- User-friendly web interface

## Technologies Used

- PHP
- HTML
- CSS
- JavaScript

## Learning Content

The platform provides educational content covering:

- Introduction to Artificial Intelligence
- Applications of AI in daily life
- Benefits and challenges of AI
- Interactive multimedia learning materials
- Knowledge assessment through quizzes

## Purpose

The purpose of this project is to enhance users' understanding of Artificial Intelligence through an interactive and engaging web-based learning experience.
